RogueVault is a community-driven discovery and rating platform built specifically for fans of roguelike and roguelite games. The platform allows players to browse a curated collection of games, search by title, filter by gameplay tags, view detailed information about each title, and submit new game entries to grow the collection. RogueVault targets dedicated players who want to explore the roguelike genre and discover games matched to their preferred play style, whether that is deck-building, dungeon crawling, turn-based strategy, or fast-paced action. The goal of the project is to deliver a polished, responsive single-page web application that feels native to the gaming community it serves, using a dark roguelike-inspired visual theme throughout. All functionality runs entirely on the client side, meaning the application requires no backend servers, no external APIs, no databases, and no user accounts of any kind. The entire application is built using only HTML5, CSS3, and vanilla JavaScript, with no third-party frameworks or libraries permitted. Game entries submitted by users are persisted across sessions using the browser's built-in localStorage, ensuring a lightweight and self-contained experience. This makes RogueVault instantly runnable by opening a single HTML file in any modern web browser without any installation or network dependency.
